# Portfólio Pessoal - Yves Esteves 

Um portfólio web moderno, interativo e totalmente responsivo, desenvolvido para apresentar minha trajetória acadêmica na área de Ciência da Computação, exibir meus projetos práticos e facilitar o contato profissional.

## Funcionalidades

- **Apresentação Pessoal:** Resumo da minha formação técnica e graduação atual (UFU).
- **Vitrine de Projetos:** Exibição de projetos práticos em formato de *cards*, como o Sistema de Votação em Blockchain e a Apple Watch Product Page, com links diretos para os respectivos repositórios.
- **Integração com WhatsApp:** Formulário de contato dinâmico criado com JavaScript que captura os dados do usuário, formata a mensagem e redireciona automaticamente para o WhatsApp.
- **Design Responsivo:** Layout fluido que se adapta perfeitamente a dispositivos móveis (smartphones e tablets) e desktops, utilizando Flexbox e CSS Grid.
- **UI/UX Moderna:** Interface no estilo *Dark Mode* com uma paleta de cores "Verde Floresta/Tech", efeito *Glassmorphism* (fundo de vidro) e animações suaves de interação.

## Tecnologias Utilizadas

- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Estilização customizada, variáveis globais (`:root`), animações com `@keyframes` e *Media Queries* para responsividade.
- **JavaScript (Vanilla):** Lógica de validação de formulário e manipulação de URLs.
- **Vercel:** Hospedagem com *Continuous Deployment* (Deploy automático a cada *push* no GitHub).
